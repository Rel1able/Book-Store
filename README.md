
# Game Store App

A modern **React + TypeScript** web app for browsing and exploring games.  
This project uses **TailwindCSS** for styling, stores data in **local storage**, and fetches game information from the **RAWG API**.

---

## Features

- Browse and search games using the RAWG API
- Store favorite games locally using browser local storage
- Clean, responsive UI built with TailwindCSS
- Client-side routing with React Router
- Interactive components with React and TypeScript

---

## Technologies Used

- **React 19** + **TypeScript**
- **TailwindCSS 4**
- **Vite** for development and build
- **React Router v7**
- **ESLint** for code linting
- **RAWG API** for game data

---

## Installation

1. **Clone the repository:**
```bash
git clone git@github.com:Rel1able/Game-Store.git
cd game-store
```
2.**Install dependencies**
```bash
npm install
```
3.**Set up environment variables**
```bash
#Create a .env file in the root directory and add your RAWG API key:
VITE_RAWG_API_KEY=your_rawg_api_key_here
```
You can obtain a free RAWG API key here: https://rawg.io/apidocs
